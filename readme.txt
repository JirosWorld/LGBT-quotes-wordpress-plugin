Upload the zipped plugin files to the `/wp-content/plugins/rainbow-queery` directory, or install the plugin through the WordPress plugins screen directly.2. Activate the plugin through the 'Plugins' screen in WordPress== Frequently Asked Questions === What do you hope to achieve with this plugin? =My big dream is to make all technology and all databases more diversity friendly, one plug-in at a time.= Can I suggest additional quotes to be added to future versions? =Of course!
